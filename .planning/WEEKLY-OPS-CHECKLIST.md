# Weekly Operations Checklist

## When You Publish a New Dossier

**ALWAYS do this after publishing:**

1. [ ] Push to GitHub (triggers Vercel deploy)
2. [ ] Wait for deploy to complete (~2 min)
3. [ ] Verify page loads on pepcodex.com
4. [ ] **Send email to subscribers via Beehiiv:**
   - Subject: "New: [Peptide Name] Research Dossier"
   - Body: 2-3 sentence hook + link to dossier
   - Keep it short — they'll click through to read
5. [ ] Post to Instagram (carousel or story)
6. [ ] Update sitemap in Search Console (optional, auto-updates)
