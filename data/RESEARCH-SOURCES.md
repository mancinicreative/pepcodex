## Search Strategy

### Step 1: Primary Search
1. Search PubMed with peptide name + key terms
2. Check ClinicalTrials.gov for registered trials
3. Search Europe PMC for European sources

### Step 2: Specialized Journals
4. Search Tier 3 peptide-specific journals directly
5. Check relevant therapeutic area journals (Tier 4)

### Step 3: International Sources
6. Search Chinese databases (CNKI, Wanfang) via English interfaces
7. Check Japanese journals (J-STAGE)
8. Search Russian literature when relevant (especially for Selank, Semax, Epithalon, Cerebrolysin)

### Step 4: Key Researchers
9. Search publications by key researchers in the field
10. Check lab websites for preprints and recent work

### Step 5: Structural/Chemical Data
11. Check PDB for crystal structures
12. Search ChEMBL/DrugBank for pharmacology data
13. Review UniProt for sequence information
