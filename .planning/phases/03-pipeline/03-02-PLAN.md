<task type="auto">
  <name>Task 1: Create banned content scanner</name>
  <files>scripts/qa-banned-content.js</files>
  <action>
Create Node.js script that scans MDX content for banned patterns:

**Banned Patterns (BLOCK - fail QA):**
```javascript
const BANNED = [
  /\b(inject(ion)?s?|dose|dosing|dosage|protocol|cycle|stack(ing)?)\b/gi,
  /\b(buy|purchase|source|vendor|supplier|where to (get|buy))\b/gi,
  /\b(how to (use|take|administer|inject))\b/gi,
  /\d+\s*(mg|mcg|iu|units?)\s*(per|daily|weekly|twice|once)/gi,
  /\b(subcutaneous(ly)?|intramuscular(ly)?|iv|intravenous(ly)?)\s+(inject|admin)/gi
];
```

**Overclaim Patterns (WARN - flag for review):**
```javascript
const OVERCLAIMS = [
  /\b(proven|guaranteed|cures?|definite(ly)?)\b/gi,
  /\b(100%|always works?|miracle|amazing results?)\b/gi,
  /\b(safe for everyone|no side effects?)\b/gi
];
```

Export function `scanBannedContent(mdxContent)` that returns:
```javascript
{
  passed: boolean,
  banned: [{pattern: string, matches: string[], line: number}],
  warnings: [{pattern: string, matches: string[], line: number}]
}
```

CLI: `node scripts/qa-banned-content.js <mdx-file>`
  </action>
  <verify>Script correctly flags test content: `echo "inject 10mg daily" | node scripts/qa-banned-content.js --stdin` fails</verify>
  <done>Scanner catches all banned patterns and returns structured results</done>
</task>

<task type="auto">
  <name>Task 3: Create evidence labeling checker</name>
  <files>scripts/qa-evidence-labels.js</files>
  <action>
Create Node.js script that ensures preclinical evidence is properly labeled:

**Required Labels for Preclinical:**
- Animal studies must include: "in mice", "in rats", "rodent", "animal model", "preclinical"
- In-vitro studies must include: "in vitro", "cell culture", "cell line"

**Check Logic:**
1. Find sentences/paragraphs discussing evidence
2. Cross-reference with source pack: sources with `subjects: "animal"` or `subjects: "in-vitro"`
3. If source is referenced but text doesn't include appropriate label, flag it

Export function `checkEvidenceLabels(mdxContent, sourcePackPath)` returning:
```javascript
{
  passed: boolean,
  issues: [{
    sourceId: "PMID:xxx",
    subjects: "animal",
    context: "...sentence where it's referenced...",
    missingLabel: "animal study indicator"
  }]
}
```

CLI: `node scripts/qa-evidence-labels.js <mdx-file> <source-pack.json>`
