<task type="auto">
  <name>Task 1: Create Batch Orchestrator n8n workflow</name>
  <files>n8n/workflows/batch-orchestrator.json, n8n/docs/batch-orchestrator.md</files>
  <action>
Create n8n workflow that orchestrates full pipeline for multiple peptides.

**Workflow Structure:**

1. **Manual Trigger**
   - Input: peptides (array of {name, aliases})
   - Example: [{name: "Tirzepatide", aliases: ["Mounjaro"]}, ...]

2. **Split Into Items** (Split In Batches node)
   - Process one peptide at a time
   - Batch size: 1 (sequential to respect rate limits)

3. **Execute: Source Pack Builder** (Execute Workflow node)
   - Call source-pack-builder workflow
   - Pass peptide name and aliases
   - Get source pack JSON

4. **Execute: Draft Generator** (Execute Workflow node)
   - Call draft-generator workflow
   - Pass source pack
   - Get MDX draft

5. **Execute: QA Gate** (Execute Workflow node)
   - Call qa-gate workflow
   - Pass MDX and source pack
   - Get pass/fail

6. **Branch: QA Result** (IF node)
   - If passed: continue to publisher
   - If failed: log failure, skip to next peptide

7. **Execute: Publisher** (Execute Workflow node)
   - Call publisher workflow
   - Pass MDX and slug
   - Get publish confirmation

8. **Wait Node** (Wait node)
   - Wait 5 seconds between peptides
   - Respect API rate limits

9. **Aggregate Results** (Code node at end)
   - Collect: processed, passed, failed, published
   - Return summary

**Error Handling:**
- If any sub-workflow fails, log error and continue to next peptide
- Don't stop batch on single failure

**Documentation:**
- How to run batch
- Rate limit considerations
- Monitoring progress
- Handling failures
  </action>
  <verify>Workflow JSON is valid and includes all sub-workflow calls</verify>
  <done>Batch orchestrator chains all workflows with error handling</done>
</task>
