## Task 2: Configure GA4 Enhanced Events
**type:** auto
**gate:** blocking

**Action:**
1. Create `src/scripts/analytics.ts` with event helper functions
2. Add custom events:
   - `comparison_view` — fire on comparison page load (slug, peptides)
   - `dossier_view` — fire on dossier page load (slug, category, evidence_level)
   - `calculator_use` — fire on calculator interaction (type, peptide)
   - `search_query` — fire on Pagefind search (query text)
   - `newsletter_signup` — fire on successful subscription (source page)
   - `outbound_link` — fire on external citation clicks (url, source_type)
3. Import analytics script in BaseLayout.astro
4. Add event listeners to relevant components

**Verify:** GA4 Realtime report shows custom events firing

---

## Task 3: Set Up GA4 Conversion Funnels
**type:** checkpoint:human-action
**gate:** blocking

**Instructions:**
1. Go to GA4 Admin → Events → mark `newsletter_signup` as conversion
2. Create audience segments:
   - Comparison visitors (viewed 2+ comparison pages)
   - Dossier deep readers (time on page > 2 min on dossier)
   - Calculator users (used any calculator)
3. Create exploration report: newsletter signup funnel (landing → comparison/dossier → signup)

**Done:** GA4 shows conversion tracking and funnel report configured

---

## Task 4: Export Search Console Data
**type:** checkpoint:human-action
**gate:** blocking

**Instructions:**
1. Go to Google Search Console → Performance
2. Set date range: last 3 months (or since first indexing)
3. Export:
   - Top queries by impressions (all)
   - Top queries by clicks (all)
   - Top pages by clicks (all)
   - Top pages by impressions (all)
4. Note: Total indexed pages from Coverage report
5. Note: Any indexing errors or exclusions
6. Save data to `.planning/search-console-export/`

**Done:** CSV exports saved, summary prepared
