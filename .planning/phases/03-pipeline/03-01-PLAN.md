<task type="auto">
  <name>Task 1: Create source pack JSON schema</name>
  <files>data/schemas/source-pack.schema.json</files>
  <action>
Create JSON Schema (draft-07) defining source pack structure:

```json
{
  "peptide": {
    "name": "string (required)",
    "aliases": ["array of strings"],
    "category": "string (metabolic|repair-recovery|hormonal|cognitive|other)"
  },
  "sources": [
    {
      "id": "string (PMID:xxx, DOI:xxx, or NCT:xxx) - required",
      "type": "pubmed|europePMC|clinicalTrials|preprint",
      "title": "string (required)",
      "authors": ["array of strings"],
      "journal": "string",
      "year": "number (required)",
      "studyType": "RCT|observational|meta-analysis|review|case-series|preclinical|in-vitro",
      "subjects": "human|animal|in-vitro",
      "sampleSize": "number or null",
      "openAccess": "boolean",
      "url": "string (full URL to source)",
      "abstract": "string (optional, for context)"
    }
  ],
  "trials": [
    {
      "nctId": "string (NCTxxxxxxxx)",
      "title": "string",
      "status": "recruiting|active|completed|terminated|withdrawn",
      "phase": "1|2|3|4|not-applicable",
      "conditions": ["array of strings"],
      "interventions": ["array of strings"],
      "enrollmentTarget": "number",
      "startDate": "ISO date string",
      "completionDate": "ISO date string or null"
    }
  ],
  "metadata": {
    "generatedAt": "ISO datetime",
    "sourceCounts": {
      "total": "number",
      "human": "number",
      "preclinical": "number",
      "openAccess": "number"
    }
  }
}
```

Include JSON Schema validation keywords: required, enum, pattern for IDs (PMID:\d+, NCT\d+, DOI:.+).
  </action>
  <verify>JSON Schema is valid: `node -e "require('./data/schemas/source-pack.schema.json')"`</verify>
  <done>Schema file exists at data/schemas/source-pack.schema.json with all required fields</done>
</task>

  <action>
Create Node.js script using ajv for JSON Schema validation:

1. Import ajv and the schema
2. Export function `validateSourcePack(packPath)` that:
   - Reads JSON file from path
   - Validates against schema
   - Returns {valid: boolean, errors: array}
3. Add CLI support: `node scripts/validate-source-pack.js <path>`
4. Pretty-print errors with field paths
5. Check for duplicate source IDs (PMID/DOI/NCT should be unique)
6. Exit code 0 if valid, 1 if invalid

Install ajv if not present: `npm install ajv --save-dev`
