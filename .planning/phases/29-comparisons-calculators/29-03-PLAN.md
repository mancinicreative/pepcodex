1. **getStaticPaths**: Query the peptides collection, read `data/calculator-presets.json`, and generate paths for the 50 peptides that have presets:
```typescript
import { getCollection } from 'astro:content';
import { readFileSync } from 'fs';

export async function getStaticPaths() {
  const presets = JSON.parse(readFileSync('data/calculator-presets.json', 'utf-8'));
  const peptides = await getCollection('peptides');

  return presets.map(preset => {
    const peptide = peptides.find(p => p.slug === preset.slug);
    return {
      params: { peptide: preset.slug },
      props: {
        peptide,
        preset,
      },
    };
  }).filter(entry => entry.props.peptide); // Only include if peptide exists in collection
}
```

2. **SSR fallback**: Support both static and SSR modes using the standard pattern:
```typescript
let peptide = Astro.props.peptide;
let preset = Astro.props.preset;
if (!peptide) {
  const { peptide: slug } = Astro.params;
  const { getEntry } = await import('astro:content');
  peptide = await getEntry('peptides', slug);
  const presets = JSON.parse(readFileSync('data/calculator-presets.json', 'utf-8'));
  preset = presets.find(p => p.slug === slug);
}
if (!peptide) return Astro.redirect('/calculator/reconstitution');
```

3. **Page content**: Re-use the EXACT same calculator HTML and `<script>` from the existing `reconstitution.astro`, but with these changes:
   - Title: `${peptide.data.name} Reconstitution Calculator | PepCodex`
   - Description: `Calculate reconstitution volumes for ${peptide.data.name}. Pre-filled with typical ${preset.reconstitution.typicalVialSizeMg}mg vial values.`
   - Breadcrumb: Calculators > Reconstitution > {peptide.data.name}
   - Pre-fill input default values from preset:

     - `waterVolume` input value = `preset.reconstitution.commonWaterVolumeMl`

   - Add a "Back to Generic Calculator" link: `<a href="/calculator/reconstitution">Use generic calculator</a>`
   - Add a "View {peptide.data.name} Dossier" link: `<a href="/peptides/${peptide.slug}">View full dossier</a>`
   - Include HowToSchema with peptide-specific name: `How to Calculate ${peptide.data.name} Reconstitution`
   - Add a note about the preset: `<p>Pre-filled with typical ${peptide.data.name} vial size (${preset.reconstitution.typicalVialSizeMg}mg). ${preset.reconstitution.notes}</p>`
