# Regulatory & Research Integrity Sources

Comprehensive guide to regulatory databases and research integrity resources for peptide and supplement research.

---

## FDA Drugs@FDA

**URL:** https://www.accessdata.fda.gov/scripts/cder/daf/
**Trust:** 5/5 | **Access:** Free | **Coverage:** All FDA-approved drugs

### What It Is
FDA's official database of approved drug products. Contains drug labels (prescribing information), approval letters, reviews, and medication guides for all approved drugs including peptide therapeutics.

### Key Information Types
- Full prescribing information (label)
- Approval history and dates
- FDA review documents
- Chemistry reviews
- Clinical pharmacology reviews
- Risk evaluation documents

### Example Searches
```
Semaglutide -> Ozempic/Wegovy labels, approval packages
Tirzepatide -> Mounjaro full prescribing info
Insulin glargine -> All formulations, biosimilars
Glucagon -> Emergency formulations
Exenatide -> Byetta/Bydureon reviews
```

### When to Use
- Verify FDA approval status
- Access official prescribing information
- Review clinical data from approval packages
- Check labeled indications vs off-label claims
- Understand safety warnings and contraindications

### Tips
- Download full approval packages for detailed clinical data
- Check "Label" tab for current prescribing information
- Review "Letters" for approval conditions
- Medical reviews contain efficacy/safety summaries
- Chemistry reviews have formulation details

### Trust Rationale
Official FDA source. Regulatory-grade data. Gold standard for US drug approval verification.

## FDA FAERS (Adverse Event Reporting System)

**URL:** https://www.fda.gov/drugs/questions-and-answers-fdas-adverse-event-reporting-system-faers
**Dashboard:** https://fis.fda.gov/extensions/FPD-QDE-FAERS/FPD-QDE-FAERS.html
**Trust:** 4/5 | **Access:** Free | **Coverage:** Voluntary adverse event reports

### What It Is
FDA's database of adverse event and medication error reports. Contains voluntary reports from healthcare professionals, consumers, and manufacturers. Useful for signal detection but requires careful interpretation.

### Key Caveats
- Reports are unverified (not proven causal)
- Voluntary reporting (underreporting common)
- Reporting rates vary by drug visibility
- Cannot determine incidence rates
- Biased by publicity/litigation

### Query Approaches
- Search by drug name
- Filter by reaction type
- Review case narratives
- Compare signal strength across drugs

### When to Use
- Explore reported adverse events
- Identify safety signals
- Understand reported reaction types
- Contextualize case reports
- Research rare adverse events

### Tips
- Never cite raw FAERS counts as evidence
- Use for hypothesis generation only
- Compare relative reporting patterns
- Check for confounders (indication bias)
- Cross-reference with published pharmacovigilance studies

### Trust Rationale
Official FDA data but unverified voluntary reports. Useful for signal detection, not proof of causation. Moderate trust for exploratory research.

### Key Information Types
- Approved drug products
- Therapeutic equivalence ratings
- Patent information
- Exclusivity periods
- Applicant holder information

### Example Searches
```
Semaglutide -> Patent/exclusivity status
Insulin -> All approved formulations, biosimilars
Glucagon -> Approved products, equivalence ratings
```

### When to Use
- Verify drug approval status
- Identify generic/biosimilar availability
- Check patent expiration dates
- Research therapeutic equivalence
- Understand market exclusivity

### Tips
- "AB" rating = therapeutically equivalent
- Check both active ingredient and formulation
- Review patent expiration for biosimilar timeline
- Use for regulatory intelligence

### Trust Rationale
Official FDA regulatory document. Legal authority for therapeutic equivalence determinations.

## EMA EPARs (European Public Assessment Reports)

**URL:** https://www.ema.europa.eu/en/medicines
**Trust:** 5/5 | **Access:** Free | **Coverage:** Centrally authorized EU medicines

### What It Is
European Medicines Agency's public assessment reports for medicines authorized via centralized procedure. Contains detailed scientific evaluation, clinical data summaries, and risk-benefit assessments.

### Key Document Types
- Product information (SmPC, PIL)
- Scientific assessment reports
- Risk management plans
- Procedural steps/scientific information
- Post-authorization updates

### Example Searches
```
Semaglutide -> Ozempic/Wegovy EPAR
Tirzepatide -> Mounjaro assessment report
Insulin biosimilars -> Full clinical packages
```

### When to Use
- Access detailed clinical trial data
- Review EMA scientific assessment
- Compare EU vs FDA approval basis
- Understand European indication differences
- Research post-marketing requirements

### Tips
- EPARs often contain more clinical detail than FDA
- Check "Assessment history" for updates
- Review "Risk management plan" for safety
- Download "Product information" for official label
- Search by active substance or brand name

### Trust Rationale
Official EMA regulatory documents. Equivalent authority to FDA for EU market. Comprehensive scientific assessments.

## Retraction Watch Database

**URL:** https://retractionwatch.com
**Database:** http://retractiondatabase.org
**Trust:** 5/5 | **Access:** Free | **Coverage:** 40K+ retracted papers

### What It Is
Comprehensive database of retracted scientific papers. Essential for verifying research integrity before citing peptide studies.

### Key Features
- Searchable by author, journal, subject
- Retraction reasons documented
- Links to original papers
- Retraction notices
- Expressions of concern

### Search Examples
```
peptide -> Retracted peptide papers
Khavinson -> Check author's retraction history
[Journal name] -> Journal's retraction record
BPC-157 -> Any retracted studies
bioregulator -> Related retractions
```

### When to Use
- Before citing any research
- Evaluating author credibility
- Checking journal integrity
- Investigating suspicious findings
- Due diligence on controversial claims

### Red Flags to Check
- Duplicate publication
- Data fabrication/falsification
- Plagiarism
- Image manipulation
- Undisclosed conflicts
- Ethical violations

### Tips
- Always search before citing key papers
- Check prolific authors in niche fields
- Note "expressions of concern" (pre-retraction)
- Review retraction reasons for patterns
- Cross-reference with PubPeer

### Trust Rationale
Industry-standard integrity resource. Maintained by experienced science journalists. Comprehensive and regularly updated.

## PubPeer

**URL:** https://pubpeer.com
**Browser Extension:** Available for Chrome/Firefox
**Trust:** 4/5 | **Access:** Free | **Coverage:** Post-publication peer review

### What It Is
Platform for post-publication peer review and scientific discussion. Users (often anonymous) flag potential issues with published papers including data problems, methodology concerns, and image manipulation.

### Key Features
- Post-publication comments on papers
- Image forensics discussions
- Statistical analysis critiques
- Methodology concerns
- Author responses (sometimes)

### Search Examples
```
[DOI] -> Comments on specific paper
[Author name] -> All flagged papers by author
peptide bioregulator -> Flagged papers in field
[Journal name] -> Papers with comments
```

### When to Use
- Evaluating controversial papers
- Checking for known issues before citing
- Investigating unusual claims
- Due diligence on key studies
- Understanding scientific debates

### Interpreting Comments
- Not all comments indicate problems
- Anonymous comments vary in quality
- Author responses provide context
- Multiple independent concerns more significant
- Check if issues were addressed/corrected

### Tips
- Install browser extension for automatic alerts
- Check papers before relying on them
- Read full discussion threads
- Note if authors responded
- Cross-reference with Retraction Watch

### Trust Rationale
Valuable crowdsourced integrity checking. Quality of comments varies. Use as flag for further investigation, not definitive judgment.

## Search Strategy: Regulatory Due Diligence

### For Approved Peptide Drugs
1. **Drugs@FDA** - Verify US approval, get label
2. **EMA EPARs** - European assessment, clinical data
3. **Orange Book** - Patent/exclusivity status
4. **Health Canada/TGA/PMDA** - Global status

### For Research Peptides (Unapproved)
1. **FDA Warning Letters** - Check for enforcement actions
2. **FDA Enforcement Reports** - Recalls, tainted products
3. **Retraction Watch** - Verify cited research
4. **PubPeer** - Check for data concerns

### For Supplements
1. **NIH ODS Fact Sheets** - Baseline evidence
2. **Natural Medicines** - Professional assessment
3. **DSLD** - Market products containing
4. **FDA Tainted Supplements** - Safety alerts
5. **DSID** - Quality/variability data

### For Research Integrity
1. **Retraction Watch** - Search author/topic
2. **PubPeer** - Check specific papers
3. **COPE Guidelines** - Understand standards
4. **Cross-reference** - Compare multiple sources
